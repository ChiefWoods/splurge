# App

## Setup

1. Install dependencies

```bash
bun i
```

2. Configure `.env` files

```bash
cp .env.example .env.development; cp .env.example .env.production
```

3. Start development server

```bash
bun run dev
```

4. Build project

```bash
bun run build
```

5. Start production mode

```bash
bun run start
```
